**GhiblifyAI** is a full-stack AI web application that transforms user-uploaded images into **Studio Ghibli-inspired artwork** using the **Stability AI API**. The project combines a **React frontend** with a **Spring Boot backend** to provide a smooth and interactive image generation experience.

---

## ✨ Features

- 🖼️ Upload images for AI transformation
- 🎨 Generate Studio Ghibli-inspired artwork
- ⚡ Fast backend powered by Spring Boot
- 🌐 Clean and responsive React UI
- 🔗 Stability AI API integration

---

## 🛠️ Tech Stack

- **Frontend:** React
- **Backend:** Java, Spring Boot
- **AI Model:** Stability AI API
- **Build Tool:** Maven
- **Package Manager:** npm

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/dj-ayush/GhiblifyAI.git
cd GhiblifyAI
```

### Backend

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm start
```
